
# PALS HUB SERVER

This project is under developement


![Logo](https://raw.githubusercontent.com/itxsoumya/images/main/palshubgit.png)


## Setup on Local System


```bash
  git clone git@github.com:itxsoumya/pals_hub_server.git
  npm install
  cd pals_hub_server
```
## Run

```bash
  npm start
```
